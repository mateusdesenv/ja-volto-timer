# Ja Volto Timer

Tela estatica de pausa para live com contagem regressiva.

## Uso

Abra a pagina e informe o tempo no prompt, ou passe o tempo pela URL:

```text
/?min=7
```

Formatos aceitos:

- `7` para sete minutos
- `1:30` para um minuto e trinta segundos
- `01:00:00` para uma hora

Pressione `R` para reiniciar o timer.

## Deploy

Projeto estatico publicado na Vercel a partir do `index.html`.
